# yb-python
python 기초 공부

## 파이썬 명령어
```sh
# version확인
python --version

# pythone IDLE 접속
python

# hello_world.py 파일 실행
$ python hello_world.py
```

## 파이썬 가상환경

```sh
# 가상환경 생성
$ python -m venv cha

# 가상환경 활성화 (mac/linux)
$ source cha/bin/activate

# 가상환경에서 외부라이브러리 설치 해보기
$ pip install requests

```
